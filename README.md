### 适合人群
1. 1-3年的Java工程师
2. 缺乏项目经历的IT工作者
3. 想从面试脱颖而出的
4. 应届毕业生或者求职者

### 技术储备
1. Java体系基础知识，SSM框架
2. 理解Spring的思想和理念
3. 前端基础，SQL语法
4. Web项目经验
